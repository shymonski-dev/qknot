Q-Knot launchers

These launchers start Q-Knot without requiring Node on the local machine.

Requirements:
- Python 3.10, 3.11, or 3.12
- Frontend distribution files in the dist folder

Launch:
- macOS: ./launchers/start-macos.command
- Linux: ./launchers/start-linux.sh
- Windows: .\launchers\start-windows.bat
